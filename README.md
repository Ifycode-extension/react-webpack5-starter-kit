# React starter kit (LESS)

## Development setup
This starter kit uses the following technologies:
- LESS
- Webpack 5
- Babel 7
- Eslint

## Install dependencies

````
npm install
````

## Run application

````
npm start
````

The application will open and run at http://localhost:5050/. Modify port or entire starter kit as desired.
